# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0-alpha] - 2025-12-24

### Added

- Initial public release of KaijuTranslator.
- Multi-provider support (OpenAI, DeepSeek, Gemini).
- Premium Web Dashboard for management.
- Universal Language Support (100+ languages with searchable grid).
- Zero-clutter stub generation.
- Automated SEO (Hreflang & Sitemap index).
- "Premium Widget" with glassmorphism UI.
- Professional community standards (Contributing, Security).
- Docker demo environment.

### Fixed

- Relative path calculation for stubs in root directories.
- Web-based execution for "Build Stubs" button.
- Scanner exclusion logic to prevent partial string matches.
- CLI Base URL auto-detection.
